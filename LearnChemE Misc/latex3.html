Basic control modes
Proportional-only $$ u'(t) = K_{c} \; e(t) $$ $$ u(t) = \overline{u} + K_{c} \; e(t) $$
Integral-only $$ u'(t) = \frac{1}{ \tau_{I} } \sum^{ t }_{ i = 0 } e(i) $$ $$ u(t) = \overline{u} + \frac{1}{ \tau_{I} } \sum^{ t }_{ i = 0 } e(i) $$
Proportional-integral $$ u'(t) = K_{c} \left( e(t) + \frac{ 1 }{ \tau_{I} } \sum^{ t }_{ i = 0 } e(i) \right)$$ $$ u(t) = \overline{u} + K_{c} \left( e(t) + \frac{ 1 }{ \tau_{I} } \sum^{ t }_{ i = 0 } e(i) \right) $$
\( u'(t) \)
— change to manipulated variable
\( u(t) \)
— manipulated variable
\( \overline{u} \)
— controller bias
\( K_{c} \)
— controller gain
\( \tau_{I} \)
— controller integral time
\( t \)
— time
